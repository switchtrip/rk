Surabhi Music Academy 
Frontend source code
Deployed via gh-pages on domain: surabhimusic.com


Steps to getting started with the source code

1. Install nodejs
2. Clone the project on the terminal and move into the cloned project
3. npm i #Install the node dependencies
4. npm run dev # Starts the server on the localhost
5. Check out the package.json file and spot different commands under the "scripts" tag.
