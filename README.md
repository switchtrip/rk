# 🎵 Surabhi Music Academy — Frontend

Frontend source code for the **Surabhi Music Academy** website.

🌐 **Live Website:** https://surabhimusic.com  
🚀 **Deployment:** GitHub Pages (`gh-pages`) of https://github.com/switchtrip/rk

---

## 📦 Tech Stack

The frontend is built using a modern, lightweight web development stack focused on speed, simplicity, and maintainability.

- **Node.js** — JavaScript runtime environment
- **npm** — Package manager
- **Vite** — Fast development server and build tool

- **Svelte** — Component-based UI framework
- **Skeleton** — UI toolkit and design system for Svelte

- **GitHub Pages (`gh-pages`)** — Deployment platform

---

## 🛠️ Getting Started

Follow these steps to run the project locally on your machine.

### 1. Install Node.js

Download and install Node.js if it is not already installed.

Verify installation:

```bash
node -v
npm -v
```

Both commands should return version numbers.
---

### 2. Clone the Repository

```bash
git clone https://github.com/switchtrip/rk.git
cd rk
```

---

### 3. Install Dependencies

```bash
npm install
```

---

### 4. Start the Development Server

```bash
npm run dev
```

The application will start on:

```
http://localhost:5173
```

---

## 📜 Available Scripts

All commands are defined in the `package.json` file under the `"scripts"` section.

Common commands include:

| Command | Description |
|--------|-------------|
| `npm run dev` | Starts the local development server |
| `npm run build` | Builds the project for production |
| `npm run preview` | Previews the production build locally |
| `npm run deploy` | Deploys the site to GitHub Pages |
